# Music Store 🎵

![Gif](.Music Store.gif)

A full-stack music store web application built with Node.js, Express, MongoDB, and React.

The system includes authentication and authorization using JWT, product management for admins,
shopping cart functionality for users, and a modern UI built with PrimeReact.

---

# Features

## Guest Users
- View all products
- Login and Register

## Registered Users
- Add products to cart
- Manage shopping cart

## Admin Panel
- Add new products
- Update existing products
- Delete products
- Full product management system

## UI & Experience 
- Responsive design
- Modern UI built with PrimeReact

---

# Authentication & Security

- JWT Authentication
- Protected Routes

- Secure API endpoints
- RESTful API architecture
